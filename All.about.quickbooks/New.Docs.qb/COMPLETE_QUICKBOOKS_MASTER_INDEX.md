## Part I: Foundation & Basic Processes (0-20%)

### 1. Initial Setup & Configuration {#initial-setup}

#### 1.1 Account Creation & Subscription
- **Creating Your QuickBooks Account**
  - Navigate to quickbooks.intuit.com
  - Choose between QuickBooks Online versions:
    - Simple Start ($30/month): Basic features for new businesses
    - Essentials ($55/month): Multi-user access, bill management
    - Plus ($85/month): Inventory tracking, project profitability
    - Advanced ($200/month): Custom fields, advanced reporting
  - Sign up with business email
  - Verify email and set strong password
  - Enable two-factor authentication for security

#### 1.2 Initial Company Setup Wizard
- **First-Time Setup Process**
  - Enter business name and legal entity type
  - Provide EIN/Tax ID number
  - Set fiscal year start date
  - Choose accounting method (Cash vs Accrual)
  - Select industry for customized chart of accounts
  - Set company address and contact information
  - Upload company logo for professional documents

#### 1.3 User Access & Security
- **Setting Up Users and Permissions**
  - Admin user configuration
  - Creating user roles (Accountant, Employee, Time Tracking)
  - Setting permission levels:
    - All access
    - Limited access (specific modules)
    - View-only access
  - Audit trail activation
  - Password policies and requirements

## Troubleshooting & Best Practices

### Common Issues & Solutions

#### Banking Issues
- **Connection Problems**
  - Error 103: Update bank credentials
  - Error 324: Wait 24-48 hours
  - Missing transactions: Check date range
  - Duplicate transactions: Use exclude feature
  - Wrong account mapping: Reconnect account

#### Reconciliation Issues
- **Discrepancy Resolution**
  - Beginning balance changes
  - Missing transactions investigation
  - Cleared status corrections
  - Previous reconciliation edits
  - Forced reconciliation procedures

#### Payroll Issues
- **Common Payroll Problems**
  - Tax calculation errors
  - Direct deposit failures
  - Form filing errors
  - Incorrect withholdings
  - Year-end adjustments

#### Performance Issues
- **System Optimization**
  - Browser cache clearing
  - Browser compatibility
  - Network speed testing
  - List limits management
  - Inactive record management
