In 2025, you can turn off the optional Two-Step Verification (which asks for a code every time you log in), but you cannot fully disable Multi-Factor Authentication (MFA) for unrecognized logins. 
1. What You CAN Turn Off
You can disable the feature that requires a security code for every single login session.
How to do it:
Sign in to your Intuit Account.
Select Sign in & security from the left menu.
Select 2-step verification.
Toggle the Use 2-step verification switch to Off.
Enter your password and the code sent to your phone to confirm the change. 
2. What You CANNOT Turn Off
Even with Two-Step Verification turned off, Intuit still uses standard Multi-Factor Authentication (MFA) to protect your data. 
Unrecognized Devices: QuickBooks will still send a verification code if you sign in from a new computer, different browser, or a private/incognito window.
Security Triggers: If the system detects a high-risk login (e.g., a different network or certain ad blockers), it will require a code even if you have disabled the optional 2-step setting. 
Tips to Reduce Prompts
If you find the codes annoying, try these steps to stay "verified":
"Remember this device": Check this box when logging in so QuickBooks recognizes your browser in the future.
Avoid Clearing Cookies: If you clear your browser's cache and cookies, QuickBooks will treat your computer as a "new" device and ask for a code again.
Whistlist Intuit: Add *.intuit.com to your browser's "Trusted Sites" or ad blocker's allow list to prevent security triggers.
