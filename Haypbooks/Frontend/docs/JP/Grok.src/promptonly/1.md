About the "Remember me for 30 days" toggle
This checkbox allows the user to bypass the OTP verification step for 30 days on the current device/browser. After 30 days (or if the user clears cookies/sessions), the "Confirm Your Identity" screen will appear again, requiring OTP verification once more.
(Note: Since the PIN feature has been removed, this OTP verification is now the sole second-factor step for enhanced security after login.)
