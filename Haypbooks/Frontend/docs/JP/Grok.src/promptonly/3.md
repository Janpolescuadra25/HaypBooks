

### Desired Behavior
1. **Responsive Adaptability**
   - The container must adapt to **any viewport size** (desktop, tablet, mobile).
   - It should not remain fixed at a centered `max-width` when the viewport is wide.

2. **Edge-to-Edge Expansion**
   - When the user **zooms out** or when the viewport width is **greater than or equal to 1600px**, the container should expand to fill the entire horizontal space.
   - The left edge of the container must align flush with the **left edge of the viewport**.
   - The right edge of the container must align flush with the **right edge of the viewport**.
   - No inset, margin, or reserved column should remain visible.

3. **Fallback / Automatic Detection**
   - Expansion should happen **automatically** without requiring a query parameter (e.g., `?forceWide=1`).
   - Detection logic can use:
     - `window.innerWidth` (≥1600px triggers wide mode).
     - `visualViewport.scale` or `devicePixelRatio` to detect zoom-out conditions.
   - If either condition is true, apply the edge-to-edge layout.

4. **Default Mode**
   - At normal zoom and standard viewport widths (<1600px), the container may remain centered with a `max-width` for readability.
   - This ensures text and content don’t stretch too wide on smaller screens.

### Technical Implementation Notes
- **CSS Approach**
  ```css
  .portfolio-container {
    margin: 0 auto; /* default centered */
    max-width: 1200px; /* default readable width */
  }

  @media screen and (min-width: 1600px) {
    .portfolio-container {
      max-width: 100%; /* expand fully */
      margin: 0;       /* remove centering */
    }
  }
  ```

- **JS/React Conditional Approach**
  ```tsx
  const isWide =
    window.innerWidth >= 1600 ||
    (window.visualViewport && window.visualViewport.scale < 1);

  return (
    <div
      className={`portfolio-container ${isWide ? "wide" : "centered"}`}
    >
      {children}
    </div>
  );
  ```

- **Class Behavior**
  - `.centered`: applies `max-width` and `margin: auto`.
  - `.wide`: applies `width: 100%` and removes margins.
