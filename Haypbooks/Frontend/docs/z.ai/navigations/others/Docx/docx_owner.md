PART 1: QUICK SETUP (Initial Onboarding)
Quick Setup collects essential information to activate your account in 15-30 minutes. Navigate through each step to complete initial configuration.
1.1 Quick Setup Steps
Step	Navigation & Actions	Outcome
Step 1: Account Creation	Full Name, Email, Password, Phone (optional)	Creates user account
Step 2: Company Information	Company Name, Business Type, Industry, Company Size	Sets company identity
Step 3: Region Selection	Country (US/PH/UK/EU/AU/SG/CA/IN/MX/JP), State/Province, Time Zone	Activates country modules
Step 4: Currency & Fiscal Year	Base Currency, Fiscal Year Start, Tax Year End	Sets reporting periods
Step 5: Chart of Accounts	Use Template / Import Custom / Start Blank	Establishes account structure
Step 6: First Bank Account	Bank Name, Account Type, Nickname, Opening Balance	Enables banking features
Step 7: Tax Registration	Tax Registration Numbers (optional)	Enables tax calculations
Step 8: Review & Confirm	Summary of all entered information, Terms acceptance	Activates account

1.2 Quick Setup Detail Navigation
Step 1: Account Creation
Field	Input Options	Navigation
Full Name	Text input	Required field
Email Address	Email validation	Required field, verification sent
Password	Password with strength indicator	Min 8 characters, mix of letters/numbers
Phone Number	Optional phone input	For account recovery and notifications
Profile Picture	Upload image (optional)	User avatar for account

Step 2: Company Information
Field	Input Options	Navigation
Company Name	Text input	Required field, appears on reports

Industry	Dropdown: Retail, Manufacturing, Services, Technology, etc.	Enables industry-specific features
Company Size	Dropdown: 1-10, 11-50, 51-200, 200+	Configures team features
Tax ID / EIN	Text input with validation	For tax reporting

Step 3: Region Selection
Field	Input Options	Navigation
Country	Dropdown: US, PH, UK, EU, AU, SG, CA, IN, MX, JP	Activates country-specific modules
State/Province	Dropdown based on country	Sets regional compliance
City/Municipality	Text input or dropdown	Location details
Time Zone	Auto-detected, manual override	Schedules and deadlines
Currency	Auto-populated based on country, override available	Base currency for transactions

Step 4: Currency & Fiscal Year
Field	Input Options	Navigation

Fiscal Year Start	Month selector	Defines financial year
Tax Year End	Month selector	Defines tax reporting period
Accounting Method	Cash / Accrual	Determines recognition timing

Step 5: Chart of Accounts
Option	Description	Navigation

Account Nickname	Text input	For easy identification

Connect Bank Feed	Optional: Connect for automatic imports	Enables bank feeds

Step 7: Tax Registration
Field	Input Options	Navigation

VAT/GST Number	Text input (if applicable)	For VAT/GST countries
BIR TIN [PH]	Text input with format validation	For PH tax reporting
Company Registration [UK/AU/SG/CA/IN/MX/JP]	Country-specific registration numbers	For regional compliance
Skip Option	Checkbox to skip this step	Can be added later in Setup Center

Step 8: Review & Confirm
Action	Description	Navigation
Review Summary	Display all entered information	Edit button for each section
Terms of Service	Checkbox for acceptance	Required to proceed
Privacy Policy	Checkbox for acceptance	Required to proceed
Create Account	Button to finalize	Account activation, redirect to Dashboard
