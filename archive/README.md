# archive/

This folder contains archived items moved from the repository root for tidiness.

Contents:
- `archive/patches/` — historical patches and apply scripts (use `archive/patches/README.md` for instructions).
- `archive/scripts/` — legacy root scripts; the canonical script locations are now under `Haypbooks/Backend/scripts/` and `Haypbooks/Frontend/e2e/scripts/`.

If you need to restore anything from `archive/`, use git to checkout or copy it back into the appropriate location.
